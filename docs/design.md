# DESIGN GUIDE

Style:

Modern pet store + corporate veterinary + laboratory

Reference style:

Petco style
Corporate pharma style
Clean UI

--------------------------------------------------

## COLORS

Primary: Blue
Secondary: Red
Background: White
Secondary background: Light gray

Use blue for headers
Use red for buttons
Use white for content
Use gray for sections

--------------------------------------------------

## TYPOGRAPHY

Titles:
Poppins / Inter / Montserrat

Text:
Inter / Open Sans

Large titles
Clean spacing
Readable text

--------------------------------------------------

## UI STYLE

Rounded cards
Soft shadows
Large images
Clean spacing
Modern icons
Smooth animations

No gradients overload.
No childish icons.

--------------------------------------------------

## LAYOUT RULES

Use sections.

Hero
Categories grid
Products grid
Testimonials slider
Gallery
Branches
Footer

Spacing must be large.

Use container max width.
