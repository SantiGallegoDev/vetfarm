# PROJECT REQUIREMENTS

Client: Vetfarm
Type: Corporate veterinary website with product catalog
Framework: Astro
Language: Spanish
Ecommerce: NO
Catalog only: YES

--------------------------------------------------

## OBJECTIVE

Create a modern, professional, corporate veterinary website that shows:

- Company information
- Product catalog with prices
- Categories by species
- Testimonials with before / after
- Event gallery
- Branch locations
- WhatsApp contact
- Mission / vision / history
- Corporate credibility

The website must feel:

- Professional
- Scientific
- Veterinary
- Modern
- Clean
- Trustworthy
- Corporate but friendly

NOT childish
NOT old
NOT basic

--------------------------------------------------

## WEBSITE TYPE

This is NOT ecommerce.

Users must be able to:

- See products
- See prices
- See presentations
- See categories
- Contact via WhatsApp

Users must NOT:

- Pay online
- Checkout
- Create cart
